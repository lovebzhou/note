### 创建



Vue 项目的创建有两种方式：

- 用 @vue/cli 创建的 webpack 作为构建工具的项目
- 用 create-vue 创建的 vite 作为构建工具的项目

#### CLI

```sh
# 安装
yarn global add @vue/cli

# 创建
vue create project-name

# 修改packageManager配置
vue config -s packageManager yarn
```
